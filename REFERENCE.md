# Heroku

###개요
무료로 사용할 수 있으며 간단한 프로젝트는 시범으로 돌려 볼 수 있는 PaaS 서비스를 제공한다.
git을 기반으로 동작한다.

### 배포 프로세스
 (1) heroku login
 (2) 로그인창 뜨면 로그인처리
 (3) heroku git 디렉터리로 변경된 프로젝트내용 복붙
 (4) git add .
 (5) git commit -am "commit msg"
 (6) git push heroku master
 (7) https://woorifitbenefit.herokuapp.com/
